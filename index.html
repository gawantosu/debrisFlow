HerschelBulkleyの係数には以下の値をconstant以下のtransportPropertiesに記載し，ビンガム流体とダイラント流体の性質を有する流体を設定しました．

transportModel  HerschelBulkley;
HerschelBulkleyCoeffs
  {
    k     0.005;
    n     2;
    tau0  1;//降伏応力
    nu0   10;//動粘性係数の最大値
  }
